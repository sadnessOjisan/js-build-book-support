# js-build-book-support

環境構築本のサンプルレポジトリ

```
# package install
npm install
```

```
# start
$ npx webpack-dev-server

# build
$ npx webpack

# start-storybook
$ npx start-storybook

# build-storybook
$ npx storybook

# run test
$ npx jest

# lint
npx eslint 'src/**/*.{ts,tsx}'

# format
npx prettier --write './src/**/*.{js,ts,tsx}'
```
